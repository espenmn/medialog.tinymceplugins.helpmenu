Links einfügen

Wenn Sie einen anklickbaren Verweis auf eine andere Seite erzeugen möchten, dann markieren Sie bitte das, was anklickbar werden soll (in der Regel ein oder mehrere Wörter, aber auch das Anklicken eines Bildes ist möglich) und klicken danach auf den Button für Links . Daraufhin erscheint folgendes Fenster:

Die "Link-URL" ist die Internet-Adresse der Zielseite. Diese Adresse kann verschieden aussehen. Das Feld mit den CSS-Klassen ist für fortgeschrittene Anwender und ermöglicht das optische Gestalten von Links (probieren Sie es aus!). Unter Wissenswertes gibt es weitere Informationen zur Verwendung von CSS-Klassen.

{#toc}

Links zu fremden Seiten

Die Adressen fremder Seiten beginnen meist mit einem http://, wie z.B. http://www.google.de. Eine solche Adresse können Sie direkt hier eingeben und auf [Einfügen] klicken - fertig.

{#toc}

Links zu eigenen Seiten

Bei Verweisen zu anderen (Unter-)Seiten dieser Website gibt es die Möglichkeit, einen Dateibrowser (ähnlich dem Windows-Explorer) zum Durchsuchen zu benutzen. Dazu klicken Sie bitte auf das Durchsuchen-Symbol .
