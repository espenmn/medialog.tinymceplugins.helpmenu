Insert/Edit Links

If you would like to create a link to another page, highlight the portion if text or the image that is to become the clickable part and click the button for insert/edit links . This will open the following dialogue:

The "Link URL" is the address of the target page. This address can look quite different. The selection field for CSS classes is for advanced users and offers additional layout possibilities for hyperlinks (try it!). Useful Information will tell you more about the use of CSS classes.

{#toc}

Links to foreign web sites

Addresses of foreign web sites mostly start with http://, like in http://www.google.de. You can enter such an address and click [Insert] - done.

{#toc}

Links to My Pages

If you want to create a hyperlink to one of your pages (or other pages on this web site) you can call the file browser (similar to Windows Explorer). Just click the browse button .
